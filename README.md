# TesteFocusTextil


## Para rodar a aplicação
- Clone o repositorio
- Execute npm update
```
npm update
```

### Iniciar o servidor
- Execute npm start
```
npm start
```

### Inciar Teste
- Execute npm test
```
npm test
```

### Parametros da rota Register
- fullname:string
- email:string
- password:string

### Parametros da rota Login
- email:string
- password:string
